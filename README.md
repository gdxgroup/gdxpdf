## Requirements

- JDK 21 or higher
- iText 9

## Installation

Steps to install the project and its dependencies:

1. Clone this repository:
   ```bash
   git clone https://github.com/gdxgroup/gdxpdf.git
   ```

2. Navigate to the project directory:
   ```bash
   cd gdxpdf
   ```

3. Install dependencies (if needed):
   ```bash
   ./mvnw install
   ```

4. Run the application (adjust according to your project type):
   ```bash
   ./mvnw spring-boot:run
   ```
